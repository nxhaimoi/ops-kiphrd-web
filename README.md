# OPS x KIPHRD Training Program Discussion Website

## Files

- `index.html` - the presentation-style webpage.
- `ops_kiphrd_layout.js` - the single editable layout/content file.
- `assets/logo-evngenco3-ops.png` - OPS logo used by the page.
- `assets/pm3-online.jpg` - PM3 image downloaded from an online public source.
- `assets/pm22-equipment-brief-source.png` - PM2.2 image/details from the provided PM2.2 brief; the online source link is kept in the slide.

## How to use

1. Open `index.html` in a browser, or use the local preview URL if a server is running.
2. Click each tab on the left to navigate.
3. Edit `ops_kiphrd_layout.js` to update content.
4. Refresh `index.html` after saving the layout file.
5. To add the PM2.2 image, set the `image` field of the PM2.2 plant card in `ops_kiphrd_layout.js`.

## Notes

- The webpage is designed as 16:9 slide-like tabs.
- The visual style uses a bright Material Design-inspired look.
- Content is written in simple English for discussion with KIPHRD.
- Current tab structure: OPS, PM3/PM2.2, Position & Competency, Roadmap, Training Domains, Discussion, Sources.
- Please verify contractual wording and internal names before using the content as a formal document.
- BOT handover dates are included per meeting brief and should be checked against EVN/GENCO3 official records before formal issue.
- Training needs include OPS establishment proposal themes: data analytics, IDPP, dashboards, early warning, reliability/performance, KPI/SLA and integrated O&M coordination with EPS.
